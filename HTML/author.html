    Mô tả về trang web

    Bố cục của trang web được lấy ý tưởng từ trang web vinfruits.com

    Trang web có đầy đủ tính năng như: chọn sản phẩm, thêm vào giỏ hàng,xóa khỏi giỏ hàng, tính tổng giỏ hàng, thay đổi số lượng, cập nhật lại giá

    Đăng nhập, đăng ký đã được validate, cần validate theo mesage để được đẩy vào route tiếp theo, có link hỗ trợ quay lại home ở dưới khung đăng ký/đăng nhập

    Dữ liệu trên trang web được lưu trữ trên localStorage với 2 item là: 'web_fruits_products' và 'web_fruits_detail_product'
